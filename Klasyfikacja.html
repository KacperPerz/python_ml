def shift_image(image,dx,dy):
    image = image.reshape((28,28))
    shifted_image = shift(image, [dy,dx], cval = 0, mode="constant")
    return shifted_image.reshape(-1)

Jako e niektre algorytmy (w tym SVM) mog dziaa za dugo, naley dokona redukcji wymiarowoci. tSNE grupuje podobne elementy, wic w obliczu klasyfikacji liczb sprawdzi si dobrze. Na duych zbiorach wykonuje si natomiast bardzo wolno - przyspiesz ten proces poprzedzajc go analiz PCA z zachowaniem 95% wariancji. Znacznie przyspieszy tSNE niewielkim kosztem dokadnoci. 2 wymiary posu do prezentacji wykresu.
